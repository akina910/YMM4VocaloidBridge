# Code of Conduct

このプロジェクトでは、経験、背景、立場に関係なく、安心して技術的な議論と共同作業ができる場を目指します。

## 期待する行動

- 人ではなく、コード・設計・検証結果について議論する
- 初心者にも再現可能な説明と根拠を示す
- 異なる意見を要約してから反論する
- ライセンス、プライバシー、権利者の条件を尊重する
- 誤りを見つけたときは、攻撃ではなく修正可能な形で伝える

## 許容しない行動

- 侮辱、差別、嫌がらせ、脅迫、個人攻撃
- 個人情報や非公開情報の投稿
- ライセンス認証の回避や不正利用を促す行為
- 建設的な議論を妨げる継続的な荒らし行為

メンテナーは、Issue、Pull Request、Discussion、コメントを編集・非表示・ロックし、必要に応じて参加を制限できます。問題の報告にはGitHubのReport機能を利用してください。セキュリティ上の問題は[SECURITY.md](SECURITY.md)に従ってください。
