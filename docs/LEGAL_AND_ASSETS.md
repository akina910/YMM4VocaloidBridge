# Legal, Trademarks, and User-Owned Assets

この文書はプロジェクトの開発方針であり、法的助言ではありません。

## Unofficial status

YMM4VocaloidBridgeは非公式のコミュニティプロジェクトです。Yamaha Corporation、クリプトン・フューチャー・メディア株式会社、饅頭遣い、その他の製品・権利者との提携、承認、スポンサー関係を示しません。

VOCALOID、初音ミク、YMM4その他の製品名・キャラクター名・商標は、識別と互換性説明のためにのみ使用し、各権利者に帰属します。

## User responsibility

利用者は、使用するソフトウェア、ボイスバンク、キャラクター素材を正規に取得し、それぞれの利用条件に従う必要があります。生成音声や映像の公開・商用利用条件も、各製品・素材の規約に従います。

## Repository policy

次をリポジトリとリリースに含めません。

- 製品インストーラー、DLL、ボイスバンク、認証データ
- シリアル番号、アカウント情報、ライセンスファイル
- 初音ミクなどのキャラクター画像、立ち絵、ロゴ
- 第三者の歌詞、楽曲、音声を許可なく含むテスト素材
- 製品の保護機構を回避するコードや手順

サンプル素材は、プロジェクトが新規作成した抽象的な口形画像、無音・合成テスト音、短い自作テキストだけを使用します。

## Compatibility approach

互換性は、公開プラグインAPI、標準MIDI/WAV、Windowsの公開UI Automation機能、ユーザーが通常操作できる画面を通じて実現します。暗号化された独自プロジェクト形式の解析や、プロセスメモリの改変は行いません。
